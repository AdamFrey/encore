## Quickstart

Add the necessary dependency to your project:

```clojure
Leiningen: [com.taoensso/replaceme "1.0.0-SNAPSHOT"] ; or
deps.edn:   com.taoensso/replaceme {:mvn/version "1.0.0-SNAPSHOT"}
```

And setup your namespace imports:

```clojure
(ns my-ns
  (:require [taoensso.replaceme :as replaceme]))
```

Check out the extensive [API docs][API] for more info from here!
